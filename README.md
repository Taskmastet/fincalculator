# Цена часа

Калькулятор стоимости одного часа твоего времени, оформленный как сцена
из павшего **Лордерона** в **Тирисфальских пустошах** (вселенная Warcraft):
кровавое небо, руины Капитал-Сити на горизонте, голые деревья, кладбищенская
ограда, туман и некротический свет.

**Живая версия:** https://taskmastet.github.io/fincalculator/

## Что делает

Вводишь два числа:

- **Жалованье** за период (месяц, неделя, смена — что угодно);
- **Часы службы** за тот же период.

Нажимаешь «Взвесить час» — и получаешь **стоимость одного часа**. Принимает
значения с пробелами (`90 000`) и запятой (`1500,5`); предупреждает о пустых,
отрицательных и нулевых вводах.

## Что внутри

- **Многослойная параллакс-сцена** — небо, руины собора, деревья, туман,
  кладбище и передний план из ветвей двигаются с разной глубиной за курсором
  (на телефоне — за наклоном устройства).
- **Реакция на расчёт** — оглашение цены поднимает ворон в небо, зажигает
  свет на стеле и проявляет приговор.
- **Звук пустоши** — синтезированный ветер и низкий дрон через Web Audio
  (по кнопке-руне, без аудиофайлов).
- **Настоящий шрифт** — Forum (капитель, как высеченная в камне) и Lora,
  вшиты в файл как base64. Никаких внешних запросов.
- **Single-file** — итоговый `index.html` самодостаточен: открывается где угодно.
- Доступность: адаптив до мобильных, видимый фокус, уважается
  `prefers-reduced-motion` (анимации и движение отключаются).

## Сборка

Исходник — `index.src.html`; шрифты лежат в `assets/fonts/inline-fonts.css`.
Финальный самодостаточный файл собирается так:

```bash
python3 build.py      # вшивает шрифты → index.html
```

## Запуск

Просто открой `index.html` в браузере. Либо по HTTPS через Caddy:

```bash
caddy run --config Caddyfile   # https://localhost
```

## Файлы

- `index.html` — собранная страница (деплоится на GitHub Pages);
- `index.src.html` — исходник без вшитых шрифтов;
- `build.py` — сборщик (вшивает шрифты);
- `assets/fonts/` — шрифты в виде base64-CSS;
- `Caddyfile` — конфиг локального HTTPS;
- `.github/workflows/pages.yml` — авто-деплой на GitHub Pages.
