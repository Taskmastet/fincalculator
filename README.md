# В час

Калькулятор стоимости одного часа твоего времени, оформленный в атмосфере
*Silent Hill 2* — туман, ржавчина и приглушённый свет.

## Что делает

Вводишь два числа:

- **Зарплата** за период (за месяц, неделю, смену — что угодно);
- **Количество отработанных часов** за тот же период.

Калькулятор делит одно на другое и показывает, **сколько ты получаешь за один час**.

Принимает значения с пробелами (`80 000`) и с запятой вместо точки (`1500,5`),
предупреждает о пустых, отрицательных и нулевых вводах.

## Запуск

Достаточно открыть `index.html` в браузере — весь CSS и JS встроены внутрь,
внешних файлов нет.

### По HTTPS (через Caddy)

```bash
caddy run --config Caddyfile
```

Сайт поднимется на `https://localhost` с локальным доверенным сертификатом,
который Caddy выпускает автоматически.

## Файлы

- `index.html` — вся страница: разметка, стили и логика в одном файле;
- `Caddyfile` — конфиг локального HTTPS-сервера.
